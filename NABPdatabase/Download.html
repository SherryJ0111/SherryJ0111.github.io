Full dataset:
The data file we provided contains information of each entry arranged as following:
PDB ID
the unique Protein Data Bank identifier for each interaction.
Protein chain ID
the chain identifier for the protein in the interaction.
NA chain ID
the chain identifier for the nucleic acid in the interaction.
protein_resno
conatining residue numbers for the binding site residues in the protein.
protein_resnames
containing formatted residue names of the binding site residues in the protein.
NA_resno
conatining residue numbers for the binding site nucleotide in the nucleic acid.
NA_resnames
containing formatted nucleotide names of the binding site nucleotides in the nucleic acid.
type
the interaction type(protein-DNA interaction or protein-RNA interaction).
protein_sequence
the sequence of corresponding protein chain.
NA_sequence
the sequence of the corresponding nucleic acid chain.
Uniprot_entry
the mapped Uniprot entry of the corresponding protein chain.
Classification
mmCIF keywords describing this structure.
Source
organism name of this PDB struction.
